info

Eklipse is a search engine for solar eclipses happening in the 21th century.

Searching is pretty straightforward, entering an area in the search box and hitting enter should query the database. The area can be a continent, ocean or a country.

Eklipse is a free software. The LICENSE, README and source code can be found at https://git.lh.mk/eklipse.

The repository is also being mirrored at https://github.com/hristijansk/eklipse.

Data:

Area - geographical area.

Date - date of a solar eclipse.

Time - time of greatest eclipse (UTC).

Duration - possible duration of a solar eclipse.

Magnitude - the fraction of the angular diameter of a celestial body being eclipsed.

Saros - a period of approximately 223 synodic months, that can be used to predict eclipses of the Sun and Moon.

Path width - the path width of the total or annular eclipse.
